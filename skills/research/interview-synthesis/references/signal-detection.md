# Indirect Signal Detection

> **Incident 2026-03-04 (Matthew Beckerleg):** Debrief marked Growth Mindset and Conflict Management as "NOT COVERED" because scripted questions weren't asked verbatim. Both were actually covered organically.

## The Problem

Interviewers don't always ask scripted questions verbatim. They often:
1. Acknowledge a signal organically
2. Pivot early after observing the signal
3. Cover multiple areas with one story
4. Follow conversation flow

## Transcript Search Patterns

**Before marking ANY behavioral area as "NOT COVERED," search for:**

### Interviewer Acknowledgment Patterns
```
"I like that you're talking about..."
"I'm picking up on..."
"That's exactly what I was going to ask about..."
"You've already answered my next question..."
"That covers [area] for me..."
```

### Pivot/Move-On Patterns (Signal Was Clear)
```
"Let's move on to..."
"Great, let's get into..."
"Perfect, shifting to..."
[Interviewer skips planned question]
```

### Organic Candidate Statements
```
"I like to learn..." → Growth Mindset
"I had to change my approach..." → Growth Mindset
"The customer really needed..." → Customer Obsession
"We disagreed about..." → Conflict Management
"Feedback helped me..." → Growth Mindset
```

## One Story, Multiple Signals

A single story often covers multiple areas. Credit all applicable areas, note the shared story, assess strength independently.

## Detection Workflow

1. **Direct questions** — verbatim or near-verbatim
2. **Interviewer acknowledgments** — patterns above
3. **Organic candidate statements** — demonstrated competencies
4. **Interviewer pivots** — signal already clear
5. **Only then** mark "NOT COVERED" if all passes find nothing
